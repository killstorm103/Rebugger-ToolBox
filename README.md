# Rebugger-ToolBox

Download the latest updates from Releases
Put the Installers in a USB and plug it into your ps3 or put it in your ps3's: /dev_hdd0/packages using FileZilla
On your ps3 go to your package manager > Install Package Files > (PS3 System Storage or Standard) >:
First install: Rebugger Toolbox Installer.pkg
Then mount dev_blind and then install: Rebugger Toolbox Category Installer (mount dev_blind first).pkg
